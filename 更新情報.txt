○最新版Android SDKでの変更点について

「作りながら覚えるAndroidプログラミング 改訂版」では、Android SDK r16を使用していますが、現在入手可能なAndroid SDK r20では64ビット版のJavaを検出しないため、ページ14の手順ではAndroid SDKをインストールできません。そのため、64ビット版のEclipseを使用する場合は32ビット版と64ビット版両方のJDKをインストールしてから作業を進めてください。ただし、今後公開されるバージョンのAndroid SDKでは変更される可能性があります。


○画像ダウンロードアプリの修正について

8月1日以降、本書STEP7の画像ダウンロードアプリで使用しているBing Search API 2.0は使用できなくなります。新しいBing Search APIを使用する手順については、「画像ダウンローダーアプリのプログラム修正について.txt」を参照してください。サンプルプロジェクトは新API対応版になっています。
